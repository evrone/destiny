## Getting Started
### Installation

Copy `git://github.com/evrone/destiny.git`

to `./script/plugin`  

### Usage

    Roll hexahedral dice twice
    >> Dice "2d6"
    => 9

    Roll hexahedral dice twice and add 1 to the result    
    >> Dice "2d6+1"
    => 6
    
    Roll hexahedral dice twice and add 1 to each rolling
    >> Dice "2d(6+1)"
    => 10
